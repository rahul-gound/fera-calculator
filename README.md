# FERA Calculator

A simple and efficient calculator application built to perform basic and advanced arithmetic operations.

## Features

- **Basic Arithmetic**: Addition, subtraction, multiplication, and division
- **Advanced Operations**: Percentage calculations, square roots, and more
- **History**: Keeps track of previous calculations
- **Keyboard Support**: Use your keyboard for quick input
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (v6 or higher)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/rahul-gound/fera-calculator.git
   cd fera-calculator
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the application:

   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

- Click the number buttons or use keyboard keys to input numbers.
- Use the operator buttons (`+`, `-`, `×`, `÷`) or keyboard keys (`+`, `-`, `*`, `/`) to select an operation.
- Press `=` or `Enter` to compute the result.
- Press `C` or `Escape` to clear the current input.
- Press `⌫` or `Backspace` to delete the last entered digit.
